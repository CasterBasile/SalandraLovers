# Salandra Lovers Progressive Web App

Welcome to the documentation for the Salandra Lovers Progressive Web App! This application has been developed by Castrese Basile.

## App Description

The Salandra Lovers PWA is a progressive web application developed to share events and information about Bosco della Salandra, a natural and cultural resource in the town of Marano di Napoli. The app offers features for viewing events, reading the latest news, and much more.

## Key Features

- View events in Bosco della Salandra.
- Read the latest news related to the forest and the surrounding environment.
- Read the history of the place

## Technologies Used

- HTML, CSS, and JavaScript for the app's structure, presentation, and interactivity.
- Firebase Realtime Database for retrieving and storing event data.
- Service Worker to enable offline functionality of the app.

## Installation and Usage

1. Clone or download the repository to a local folder.
2. Open the terminal in the local folder and launch le "http-server" command
3. Click on ip-link

## Code Structure

- `index.html`: Main page of the app.
- `src/css/`: Folder containing CSS files for the app's styling.
- `src/js/`: Folder containing JavaScript files for the app's logic.
- `src/images/`: Folder containing images used in the app.
- `manifest.json`: Manifest file for defining app properties.
